## incloud sdwan network delete

Delete an SD-WAN network

```
incloud sdwan network delete <id> [flags]
```

### Examples

```
  # Delete a network
  incloud sdwan network delete <id>

  # Skip confirmation
  incloud sdwan network delete <id> -y
```

### Options

```
  -h, --help   help for delete
  -y, --yes    Skip confirmation prompt
```
