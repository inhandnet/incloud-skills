## incloud knowledge

Search and query the knowledge base

### Synopsis

Search device documentation and ask questions powered by AI.

```
incloud knowledge [command]
```

### Available Commands

```
  ask         Ask a question to the knowledge base
  search      Search the knowledge base
```
